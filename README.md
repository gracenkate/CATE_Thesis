# CATE Thesis

This repository contains the code for my Master’s thesis comparing different algorithms for estimating Conditional Average Treatment Effects (CATE) using Python.  

For a full description of the methodology, results, and discussion, please refer to the PDF included in this repository.

## Tech Stack
- Python (Pandas, NumPy, scikit-learn)
- Jupyter Notebooks

## Project Structure  
- `CATE_Thesis.ipynb` – Jupyter notebooks implementing algorithms
- `CATE_Thesis.pdf` – Full thesis document  

## Key Features
- Implemented and evaluated multiple CATE estimation algorithms in Python  
- Developed a simulation framework for model comparison
